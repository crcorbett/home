# Getting Started

This guide will help you set up and start developing with the monorepo template.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Bun 1.3.5+** - JavaScript runtime and package manager
- **Node.js 20+** - Required for some tooling
- **Git** - Version control

## Quick Start

```bash
# Clone the repository
git clone https://github.com/your-org/template.git my-project
cd my-project

# Install dependencies
bun install

# Start all development servers
bun run dev
```

This will start:

- Web app at `http://localhost:3000`
- Admin app at `http://localhost:3001`
- Docs at `http://localhost:3001`
