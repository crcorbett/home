## Core Principles

### 1. Shared Code via Packages

Common functionality lives in `packages/` and is shared across apps. This eliminates code duplication and ensures consistency.

### 2. Live Types Development

TypeScript types are resolved directly from source files during development, eliminating the need to rebuild packages when making changes.

### 3. Parallel Task Execution

Turborepo orchestrates builds and development tasks, running independent operations in parallel while respecting dependencies.

### 4. AI-First Development

Built-in support for AI coding assistants with MCP servers, hooks, and context submodules.
